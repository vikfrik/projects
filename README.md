# Проекты
  Пока здесь размещёны:
  - t-invest (исследование в python + описание в pdf) - разведочный анализ данных (EDA), который был сделан на основе датасета [dano](https://dano.hse.ru/data2024) (Т-Инвестиции). В результате предобработки данных и анализа корреляций были выдвинуты основные гипотезы, которые могут быть проверены в ходе дальнейшего исследования.
  - ab_test - проект по a/b тестам: проверка гипотезы, что добавление в лендинг с оформлением дебетовой карты (https://www.tinkoff.ru/cards/debit-cards/tinkoff-black/) информации о дополнительных кэшбеках приведёт к росту конверсии из переходов на лендинг в оформление дебетовой карты.
  - health_case - отборочный кейс по аналитике от School of Analytics. Задача заключалась в анализе и выявлении закономерностей в поведении отдельных категорий клиентов по продлению подписки на сервис, помогающий вести здоровый образ жизни
